Hexapod from Scratch with RL
Harrison Bounds

GitHub

Designed, constructed, and trained a locomotion policy for a six-legged robot to compare learned gaits to traditional ones. Programmed a tripod gait with inverse kinematics and bezier curves, while the reinforcement learning used PPO (Proximal Policy Optimization) and the Genesis Simulation platform.

GitHub

GPT-2 From Scratch
Harrison Bounds, *adapted from Andrej Karpathy,

GitHub

Created GPT-2's transformer from scratch adapting the source code from Andrej Karpathy's tutorials. Trained on 2 A6000 RTX GPUs with a total training time of 4 days. Evaluated results on hellaswag data retrieving an accuracy that beat the original model.

Modern Grasping

GitHub
